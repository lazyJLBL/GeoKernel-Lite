# GeoKernel-Lite v1.0.1 - Presentation Polish

GeoKernel-Lite v1.0.1 is a presentation-focused polish release for the portfolio-ready GitHub project page.

## Highlights

- Polished the README with CI badge, project highlights, Quick Start, CLI demo, and visual debugging screenshots.
- Added six Streamlit-style screenshots under `assets/` for convex hull, segment intersection, polygon clipping, half-plane intersection, closest pair, and triangulation.
- Expanded `docs/robustness.md` with EPS policy, strict ordering vs EPS equality, orientation predicates, boundary-aware classifications, polygon normalization, half-plane clipping, triangulation validation, and known limits.
- Expanded `docs/edge_cases_report.md` with structured coverage for 56 degenerate and boundary cases.
- Updated GitHub About metadata with a portfolio-facing description and topics.
- Verified the Windows GitHub Actions workflow is green on `main`.

## Notes

- `v1.0.0` remains the first functional release.
- `v1.0.1` is the resume-ready presentation release.
- Delaunay triangulation remains experimental; the seven core algorithms remain the stable v1 feature set.
